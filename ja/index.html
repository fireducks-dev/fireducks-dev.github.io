コードを書き換えることなくpandasを高速化

pandasを利用したプログラムが遅いというお悩みをお持ちではありませんか？FireDucksはコード変更を必要とせず，高速化を実現します．プログラムのマルチスレッド化の煩雑さに頭を悩ませることなく，データ分析を高速化します．

サステナブルかつ経済的

クラウドコンピューティングのコストと環境への影響を気にされていますか？ 私たちの高速化技術により，クラウドの利用料金を削減し，同時にCO2排出も最小限に抑えます． FireDucksは環境に優しく，財布にも優しい選択です．

信頼と性能 二つを兼ね備えた革新性

FireDucksは，NECが長年に渡って磨き上げてきたスーパーコンピューターのエッセンスを注ぎ込んで開発されました． 日本製の高品質なFireDucksは信頼性と高い性能を約束します．

FireDucksの特徴

マルチスレッド ✨

FireDucksはマルチスレッド化されており，マルチコアCPUでの高速化を可能にします． More

実行時コンパイル ⚡

ライブラリの中に埋め込まれた実行時コンパイラがあなたのコードを最適化します． More

pandas API 🐼

FireDucksはpandas APIで利用できます． pandasとの違いはインポート文だけ． 使い方を新たに習得する必要はありません． Get started

そのままでもOK 🚀

pandasプログラムをそのままFireDucksで動かせます． FireDucksのインポートフック機能が自動でpandasをFireDucksに置き換えます． More

ベンチマーク 📈

FireDucksは，TPC-HとTPCx-BBベンチマークに含まれる様々なクエリで高い性能を示しています． More.
